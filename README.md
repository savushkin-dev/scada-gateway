# scada-gateway
Модуль обмена данными с контроллером

- симулятор OPC UA сервер на питон (plc)

- Spring Boot с подключением по ip с помощью Eclipce Milo.

Минимальные требования:
·	+ Обмен данными с реальными контроллерами посредством Eclipse Milo (OPC UA);

·	Объектная модель чтения и записи данных контроллеров (50мс);
·	Чтение и описание базы каналов;
·	Обработка и форматирование данных для передачи в Kafka;
·	Интеграция Spring Boot сервера в Kafka Producer.
Оптимальные требование:
·	Анализ журнала событий с помощью искусственного интеллекта;
·	Обмен данными по протоколу Mudbus, Profinet.
